# Лабораторная работа: PDO и ООП

## Описание проекта
Проект демонстрирует работу с базой данных через PDO с использованием ООП.

## Структура
- `config/database.php` - настройки БД
- `src/` - исходный код классов
- `Console.php` - консольный интерфейс  
- `index.php` - веб-интерфейс
- `users.sql` - SQL для таблицы

## Установка
1. Настройте `config/database.php`
2. Импортируйте `users.sql`
3. Запустите: `php -S localhost:8000`

## Использование
- Веб: откройте `index.php`
- Консоль: `php Console.php`
